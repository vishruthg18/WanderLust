To start the backend----->
1. Go to the server folder.
2. Open command prompt or terminal.
3. Run the command 'npm install'.
4. Run the command 'npm start'.


To start the front end----->
1. Go to the client folder.
2. Open command prompt or terminal.
3. Run the command 'npm install'.
4. Run the command 'npm start'.
